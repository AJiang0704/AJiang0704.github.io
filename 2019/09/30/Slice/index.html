Go语言学习笔记之Slice

切片(Slice)是围绕动态数组的概念设计的，它的长度并不固定，可以追加元素并会在切片容量不足时进行扩容。使用这种结构，可以更方便地管理和使用数据集合。

一、初识Slice

arr := [...]int{0,1,2,3,4,5,6,7}
s := arr[2:6]              //s的值为[2,3,4,5]
s := arr[:6]               //s的值为[0,1,2,3,4,5]
s := arr[2:]               //s的值为[2,3,4,5,6,7]
s := arr[:]                //s的值为[0,1,2,3,4,5,6,7]
func updateSlice(s []int){
    s[0] = 100             //arr原始数据也会改变
}

Slice本身没有数据，是对底层array的一个view

二、Slice的扩展

arr := [...]int{0,1,2,3,4,5,6,7}
s1 := arr[2:6]             //s1的值为[2,3,4,5]
s2 := s1[3:5]              //s2的值为[5,6]

Slice可以向后扩展，但不可以向前扩展

三、向Slice添加元素

s2 := append(s1,10)

添加元素时如果超越cap，系统会重新分配更大的底层数组（如果原来的数组不用了，会被垃圾回收掉）

四、创建Slice

//Creating slice
var s []int
for i :=0 ; i < 100 ; i++{
    s = append(s, 2 * i + 1)
}
s := []int{2,4,6,8}
s := make([]int,16)
s := make([]int,10,32)     //len = 10，cap = 32

五、拷贝Slice

//Copying slice
copy(s2,s1)
printSlice(s2)

六、从Slice删除元素

//Deleting elements from slice
s = append(s[:3],s[4:]...) //删掉第4个元素，len减1，cap不变
//Popping from front
//front := s[0]
s = s[1:]
//Popping from back
//tail := s[len(s)-1]
s = append(:len(s)-1)
